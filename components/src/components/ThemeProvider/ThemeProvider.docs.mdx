---
title: ThemeProvider
description: Wrapper for managing theme state
---

```tsx
import { ThemeProvider } from 'degen'
```

## Element

Pass in an element selector or reference to change where the theme binds. By default, the theme attaches to the `:root` element.

```tsx live=true
<ThemeProvider
  defaultMode={mode === 'light' ? 'dark' : 'light'}
  element={previewRef.current}
>
  <Text>The element mode is {mode === 'light' ? 'dark' : 'light'}.</Text>
</ThemeProvider>
```

## useTheme

Hook for accessing and setting theme-related properties.

```tsx
import { useTheme } from 'degen'
```

```tsx expand=true live=true
<Text>
  The current mode and accent are: {mode} and {accent}.
</Text>
```
